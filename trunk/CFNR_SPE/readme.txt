1. An Apache, MySQL and PHP server are required to run this Web Application.
2. Using tortoise SVN, get the copy of the source codes from Source -> Browse of cmsc128-adlet google code site.
3. Copy the CFNR_SPE folder to the www directory.
4. In your MySQL create a database named "cfnr_spe".
4. Using your MySQL server import CFNR_SPE-schema.sql to this database.
5. Using your web browser, type http://localhost/CFNR_SPE/add.php.
